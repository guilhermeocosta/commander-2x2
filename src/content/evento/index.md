# Calendário de Eventos

Participe dos torneios de Commander 2x2! Realizamos eventos mensais para testar o formato e conhecer novos jogadores.

## Próximos Eventos

Consulte a lista de próximos eventos abaixo. As informações são atualizadas regularmente.

## Como Participar

1. **Chegue cedo** - Os eventos geralmente começam com horário marcado
2. **Leve seu deck** - 100 cartas, commander no formato 2x2
3. **Conheça as regras** - Leia a seção Regras para entender as diferenças do Commander tradicional
4. **Divirta-se** - O objetivo é testar o formato e melhorar suas habilidades

## Local e Frequência

Nossos eventos são realizados mensalmente, geralmente no segundo sábado de cada mês. Confira a data específica do próximo evento na lista acima.

## Inscrição

A inscrição é feita no local do evento. Não há necessidade de inscrição prévia, mas recomendamos chegar com antecedência para garantir sua vaga.

## Premiação

A premiação varia conforme o número de participantes e o patrocínio do local. Geralmente temos produtos para os primeiros colocados e sorteios para todos os participantes.
